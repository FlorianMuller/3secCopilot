# 3sec Copitlot

## Install on your iPhone

No paid Apple Developer account is needed — the app is sideloaded with SideStore
using your free Apple ID, and new versions auto-update.

1. **Install SideStore** on your iPhone by following the
   [official guide](https://sidestore.io/) (you'll sign in with a free Apple ID).
2. **Add the app and install it** from the install page:
   👉 **https://florianmuller.github.io/3secCopilot/**
   Tap **Add to SideStore**, then install **3secs Copilot** from the source.

Once the source is added, future releases appear as in-app updates in SideStore —
no need to come back here.
